# 기술적인 한계
1. 위치기반 서비스 -> 현재 사용자의 위치를 활용하는 서비스들
2. 채팅 -> 일반 http 통신으로는 불가 웹소켓 통신을 사용해야함
3. 서비스 구현에 필요한 데이터가 있다면 해당 데이터를 우리가 api를 통해 수집할 수 있지만
api가 존재하기 않으면 구현이 불가하다.
